# 📊 WhatsApp Chat Analyzer

AI, Machine Learning & Cybersecurity powered WhatsApp Chat Analyzer built with Streamlit.

## 🚀 Features

- Upload WhatsApp Chat (.txt)
- Dashboard Analytics
- Active Users
- Timeline Analysis
- WordCloud
- Emoji Analysis
- Sentiment Analysis
- Spam Detection
- Link Analysis
- Fake Link Detection
- Phishing URL Analyzer
- PII Leak Detector
- Password & OTP Leak Detector
- Chat Risk Score
- AI Chat Summary
- Export Report
- Language Detection
- Search Messages

## 🛠️ Technologies Used

- Python
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Seaborn
- WordCloud
- NLTK
- Regex
